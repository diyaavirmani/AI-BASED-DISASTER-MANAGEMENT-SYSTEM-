Disaster Response and Resource Allocation using AI Problem: In natural disasters, response efforts are often inefficient. Solution: Predict affected zones, classify damage from satellite images, and suggest optimal resource distribution. 
\n> This repository is under active development and now includes a rudimentary severity resolver helper (see src/data_pipeline/severity_resolver.py).\n>
> ## Quickstart
> 
> 1. Create and activate a virtual environment (`python -m venv dm && source dm/Scripts/activate`).
> 2. Install requirements: `pip install -r requirements.txt`.
> 3. Run the API server: `uvicorn src.api.main:app --reload`.
> 4. Tests are available under `tests/`; execute them with `pytest`.
>
> The docker-compose file has basic web/redis services for local development.
